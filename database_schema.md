# Esquema de Banco de Dados

Atualmente o projeto **lmePortal** não possui tabelas persistidas em banco de dados relacional.
Este arquivo deve ser atualizado sempre que novas estruturas (tabelas, views, índices) forem
criados. Inclua para cada tabela:

- Nome da tabela e descrição
- Colunas (nome, tipo, restrições)
- Relações e chaves estrangeiras
- Observações relevantes (índices, triggers, políticas de segurança)

> Atualize também o arquivo `data_schema.sql` com os comandos SQL correspondentes e registre a mudança no `change_log.md`.
